# FlickrDownloader
For dowloading flickr albums or sets or whatever they call it.

#### Usage:
flicker downloader.exe [url to flickr album] [destination folder]<br>
Since I'm a lazy bastard, the url MUST look like this:<br>
https://www.flickr.com/photos/[username]/sets/[id]/<br>
And may not contain anything after the last /
